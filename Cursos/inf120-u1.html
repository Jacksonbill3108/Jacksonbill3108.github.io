1. Tipos de Datos

En Python no necesitamos declarar el tipo de variable. El intérprete lo asigna dinámicamente.


    # Ejemplo de tipado dinámico
    edad = 23          # Tipo Entero (int)
    peso = 70.5        # Tipo Flotante (float)
    nombre = "Julio"   # Tipo Cadena (str)

    print(type(edad))
