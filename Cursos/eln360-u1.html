1. Introducción a los sistemas de control

El objetivo principal de un sistema de control es mantener un proceso en las condiciones de operación deseadas, de forma segura y económica. Para lograr esto, es fundamental entender el vocabulario básico que usaremos a lo largo de toda la carrera y las dos arquitecturas principales de control.

1.1. Conceptos básicos y terminologías

Antes de diseñar un controlador, debemos definir las partes de nuestro sistema:

  • Variable controlada: Es la cantidad o condición que se mide y se controla. Suele ser la salida del sistema (por ejemplo, la temperatura de un cuarto). Al valor deseado de esta variable se le conoce como set point o punto de referencia.
  • Variable manipulada: Es la cantidad o condición que el controlador varía para afectar el valor de la variable controlada (por ejemplo, la cantidad de combustible o el voltaje aplicado).
  • Planta: Es el proceso o sistema físico que queremos controlar. Puede ser un motor, un robot, un sistema de climatización, etc.
  • Perturbación: Es cualquier influencia externa que afecta el comportamiento de la planta y que no es controlada por el sistema (por ejemplo, cambios en la temperatura ambiente, cargas variables).
  • Sensor: Dispositivo que mide la variable de interés (por ejemplo, temperatura, velocidad, posición) y proporciona retroalimentación al controlador.
  • Controlador: Algoritmo o dispositivo que procesa la información del sensor y genera una señal de control para la planta.
  • Actuador: Dispositivo que recibe la señal de control del controlador y actúa sobre la planta (por ejemplo, un motor eléctrico, una válvula).
